Experience

HTML

HTML gives the page structure. In this website, HTML is used for the navbar, headings, paragraphs, cards, images, buttons, and form.

CSS

CSS controls the layout and design. This page uses colors, spacing, flexbox, hover effects, media queries, and responsive styling.

JavaScript

JavaScript adds interaction to the page. The More Info buttons open popup sections, the close buttons hide them, and the Fun Facts section loads data.

Responsive Design

The page changes layout on smaller screens. The large navbar is replaced by a small-screen hamburger menu, and sections stack vertically.

Accessibility

Accessibility matters because websites should be usable for more people. This page includes alt text for images and aria-labels for icon-only buttons.
